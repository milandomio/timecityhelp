城镇系统

不同服务器所需建镇资金不同,请用指令/towny prices查看建镇费用.

/towny ? 城镇系统总指令菜单 /t set 或 /town ? 团队指令菜单
/t　　查看自己团队信息.
/t [团队名]　　查看指定团队信息.
/t list　　查看所有团队列表.
/t here　　查看现在站的地方的团队信息.
/tc　　进入城镇聊天频道.
/t spawn　　传送到团队主城.
/t outpost　　传送到团队前哨点.
/t deposit [数量]　　拿出指定数量的现钞捐入团队银行.
/plot claim 购买此处(16*16)城镇出售的土地,金钱流入团队银行.
/t leave　　退出团队

团队领导人指令:
/t new [名字]　　用你当前所在的地,创建1个指定名字的团队(绝对不能用中文).
/t add [名字]　　邀请指定人加入城镇.
/t kick [名字]　　请指定人退出城镇.
/t rank add [名字] assistant　　指定名字的人作为团队助理,邀请新成员加入和土地买卖等大部分权限.
/t claim　　用团队银行的钱来买断你当前所在的地(16*16大小),土地必须和原来的地相连.
/t claim outpost　　用团队银行的钱来买断你当前所在的地(16*16大小),土地作为前哨点.
/t set board [文字]　　修改团队简介为指定文字.
/t set mayor [名字]　　把团队交付给指定名字的人来领导,自己变成团员.
/t set homeblock　　把团队家设置到你当前所在的地.
/t set spawn　　把团队出生(复活)点设置到你当前所在的位置,该位置必须在团队家内.
/t set taxes [数量]　　压榨团员.
/t set name [文字]　　修改团队名称为指定文字.
/t set tag [文字]　　设定团队标签为指定文字,限制4字符.
/t toggle mobs　　切换团队领地是否刷怪.
/t toggle pvp　　切换团队领地能否互撸.
/res toggle townclaim　　进入疯狂买地模式,城镇银行只要有钱走哪买哪.
/res toggle map　　进入地图模式,走一步聊天栏显示附近领地情况.
/t claim auto　　慎用！！！以所站位置为中心,耗尽城镇银行的钱和城镇土地限购量以近圆形购买附近土地.

/plot fs [数值] 以指定价格出售所在位置土地(16*16),出售后金钱流入团队银行.
/plot nfs 设置不出售.

/t set perm　　调出权限设置列表.
权限设置:权限设置有3种,个人权限/res作用于你使用/plot claim购买的地,
地皮权限/plot由城主或地皮主人对单块地皮(16*16)设置(设置时站在要设置的地皮上),
城镇权限/t 由城主对整个城镇拥有的地皮进行设置.
使用方法为:/权限类型 set perm 对象 行为 开或关.
示例:/t set perm resident build on
此处/t对城镇设置权限 ,团员 的 建筑权限 打开.
如果不写对象则对所有对象设置,如果不写行为则对所有行为设置.
示例:/t set perm destroy off
此处/t对城镇设置权限 ,所有人破坏权限关闭(插件有效的情况下团员都无法破坏方块).

对象：resident成员 ally同盟 outsider其他人
行为：build建造 destroy破坏 switch使用按钮箱子附魔台等 itemuse吃东西喝药水
开关: on开 off关

其它设置请自行参考研究.
